У Google Docs з'явилася одна з корисних функцій Word 97

26.05.2021 09:12

Хоча сервіс Google Docs дещо поступається найсучаснішим варіантам Microsoft Word за кількістю доступних функцій, текстовим редактором можна користуватися безкоштовно і він широко застосовується – в тому числі для колективної роботи. Нещодавно у нього додали пов'язану із зображеннями функцію, яку слід було б інтегрувати ще кілька років тому.

Нова функція Google Docs

Тепер в Google Docs з'явилася можливість розміщувати зображення позаду тексту або виносити їх на передній план. Досить дивно, що Google не забезпечила такої можливості раніше - в інших текстових редакторах вона з'явилася десятиліття назад.

Якщо раніше користувачам Google Docs доводилося шукати всілякі обхідні шляхи для розміщення слів поверх зображень, то оновлення сервісу дозволяє легко розміщувати необхідний графічний контент в документах.

Важливо. Додатково це означає, що імпортовані документи Word збережуть коректне форматування.

Як вставити зображення в Google Docs

Для форматування нового типу потрібно зробити кілька простих дій:

  • натиснути правою кнопкою миші по картинці;
  • вибрати у випадаючому меню розділ "Параметри зображення";
  • у стовпці зправа викликати підрозділ "Перенесення тексту".


Функція доступна як звичайним користувачам, так і тим, хто підписаний на Google Workspace, хоча поки нова можливість доступна не усім і буде впроваджуватися поступово.
